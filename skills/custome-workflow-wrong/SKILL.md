# Workflow: Workflow Wrong

## Goal

Reset when the current implementation direction is not working.

## Process

1. Re-state problem and constraints from scratch.
2. Identify missing context and assumptions.
3. Propose 2-3 new approaches with trade-offs.
4. Recommend one approach and explain why.
5. Move to planning or implementation based on user decision.
