# Workflow: Workflow Clarify

## Goal

Resolve confusion by checking expectations against actual behavior and deciding if action is needed.

## Process

1. Restate expected behavior vs observed behavior.
2. Verify behavior using local evidence.
3. Explain findings in plain terms.
4. Classify outcome:
   - misunderstanding
   - configuration/documentation issue
   - real defect
5. If real defect, transition to planning flow.
