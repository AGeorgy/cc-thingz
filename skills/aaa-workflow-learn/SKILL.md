# Workflow: Workflow Learn

## Goal

Capture reusable project knowledge discovered during work.

## Process

1. Identify strategic patterns from session work.
2. Filter out one-off tactical details.
3. Present candidate learnings for confirmation.
4. Save accepted items to `AGENTS.md` or dedicated memory notes.

## Criteria

- Keep only information that will remain useful over time.
- Avoid duplication with existing guidance.
